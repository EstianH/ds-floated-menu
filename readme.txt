== Description ==
Add a customizable floated (movable) menu to your website.

= Features =
* Add a floating menu to any page or post.
* Enable/disable floated menu movability.
* Enable/disable menu on desktop/mobile devices.
* Customize the menu with pre-built settings or custom CSS, or both.
* Render dynamic shortcodes as menu items.
* Render the menu anywhere using a shortcode.


== Installation ==
1. Log in to your WordPress administrator account and Navigate to 'Plugins -> Add New'.
2. Input "DS Floated Menu" in the top right hand side search bar.
3. Click on the "Install Now" button for DS Floated Menu.
4. Click on the "Activate" button for DS Floated Menu.
